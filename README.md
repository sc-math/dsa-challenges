# 🧠 Problems Solutions

Repositório com minhas soluções de problemas do LeetCode, NeetCode e Beecrowd.

---

## 🚀 Sobre

Prática focada em estruturas de dados e algoritmos, com o objetivo de melhorar lógica, reconhecer padrões e evoluir para entrevistas técnicas.

---

## 📂 Organização

Problemas organizados por padrão de algoritmo e estrutura de dados, como:

- Array 
- DP (Dynamic Programming)
- Graph
- Tree
- Sliding Window
- Greedy
- Entre outros

### 🔖 Prefixos

* LC → LeetCode
* NC → NeetCode
* BC → Beecrowd

Exemplo:
LC1886.java

---

## ⚙️ Linguagens

* Java (principal)
* C
* Python

## 📈 Objetivo

* Consolidar padrões de algoritmos
* Melhorar resolução de problemas
* Preparação para entrevistas técnicas
* Revisão rápida por categoria
